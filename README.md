# Rock Paper Scissors Game

A simple and interactive Rock-Paper-Scissors game built with Python. Play against the computer with score tracking!

## Features

- **Interactive Gameplay**: Play multiple rounds against the computer
- **Score Tracking**: Keep track of wins, losses, and ties
- **Input Validation**: Handles invalid user inputs gracefully
- **Replay Option**: Choose to play again after each round
- **Simple UI**: Clean and easy-to-understand game interface

## Requirements

- Python 3.6 or higher
- No external dependencies required

## Installation

1. Clone or download this repository:
   ```bash
   git clone https://github.com/yourusername/Rock-paper-scissors-python.git
   cd Rock-paper-scissors-python
   ```

2. Ensure you have Python 3.6+ installed:
   ```bash
   python --version
   ```

## How to Play

1. Run the game:
   ```bash
   python rockPaper.py
   ```

2. Enter your choice: **rock**, **paper**, or **scissors** (case-insensitive)

3. The game will display:
   - Your choice
   - Computer's choice
   - Result (Win/Lose/Tie)
   - Current score

4. Choose whether to play again or exit

## Game Rules

- **Rock** beats **Scissors**
- **Paper** beats **Rock**
- **Scissors** beats **Paper**
- Same choice = **Tie**
